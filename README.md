# How to Run Locally


1. Clone the repository to your local machine.
    ```sh
    git clone https://github.com/bhagwan8815/codesyncfinal.git

    ```

2. Start the Backend Server

 ```sh
   cd BackEnd
   npm install
   npx nodemon

 ```

3. Start the Frontend Server

    ```sh
    cd Client
    npm install
    npm run start
 
    ```
